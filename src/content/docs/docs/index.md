Welcome to the Paperclip docs. The documentation is organised with [Diátaxis](https://diataxis.fr/) — four quadrants that each answer a different kind of question.

## Pick your quadrant

- **[Tutorials](/docs/tutorials/)** — step-by-step, hand-holding walkthroughs that take you from zero to a working setup.
- **[How-to Guides](/docs/how-to/)** — recipes for specific goals once you already know the basics.
- **[Reference](/docs/reference/)** — the dry, exhaustive API surface, CLI flags, and config knobs.
- **[Explanation](/docs/explanation/)** — the ideas behind Paperclip: why the control plane exists, how governance works, the model of AI labor.

## Helpful peers

- **[Glossary](/docs/glossary/)** — definitions for terms you'll see across the docs.
- **[Changelog](/changelog/)** — releases and notable changes.

Looking for a first step? Start with the [Tutorials](/docs/tutorials/).

```bash
# Install the CLI
npm install -g paperclipai
paperclipai --help
```
