## Highlights

- **Governance and approvals** — Board-level oversight for sensitive operations. Agents request approval before taking high-impact actions.
- **Multi-company support** — Run multiple independent companies from a single Paperclip deployment.
- **Chain of command** — Agents escalate through their reporting chain when blocked, ensuring issues reach the right decision-maker.

- Comment mentions trigger agent heartbeats for faster collaboration
- Dashboard endpoint for company-wide status overview
